# 🤖 Homework Telegram Bot

Telegram-бот для автоматического отслеживания статуса домашних работ через API Яндекс Практикума.

Бот периодически обращается к API, проверяет изменения статуса проверки домашней работы и отправляет уведомления в Telegram.

---

# Цель проекта

Проект выполнен в рамках изучения Python и работы с внешними API.

В процессе разработки проекта были применены:

* REST API;
* Telegram Bot API;
* JSON;
* HTTP-запросы;
* обработка исключений;
* логирование;
* работа с переменными окружения;
* автоматизация уведомлений;
* работа с внешними сервисами.

---

# ⚙️ Установка проекта

### Клонирование репозитория

```bash
git clone https://github.com/VirentLilium/homework-bot.git
cd homework-bot
```

### Создание виртуального окружения

```bash
python -m venv venv
```

### Активация виртуального окружения

Windows:

```bash
venv\Scripts\activate
```

Linux / macOS:

```bash
source venv/bin/activate
```

### Установка зависимостей

```bash
pip install -r requirements.txt
```

# Создание Telegram-бота

1. Откройте Telegram.
2. Найдите пользователя **@BotFather**.
3. Выполните команду:

```text
/newbot
```

4. Следуйте инструкциям BotFather.
5. Сохраните полученный токен.
6. Добавьте токен в файл `.env`.

**Файл .env не должен быть публичным!**

---

# Запуск проекта

После настройки переменных окружения выполните:

```bash
python homework.py
```

После запуска бот начнёт периодически обращаться к API и отправлять уведомления о новых результатах проверки домашних работ.

# Запуск тестов
